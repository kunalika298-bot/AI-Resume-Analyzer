# AI Resume Analyzer

An AI-powered Resume Analyzer that extracts skills from resumes and suggests job matches.

## Features
- Upload Resume PDF
- Extract Skills Automatically
- Match Resume with Job Roles
- Simple Streamlit UI

## Technologies Used
- Python
- Streamlit
- Machine Learning
- Pandas
- Scikit-learn
- PDFPlumber

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
